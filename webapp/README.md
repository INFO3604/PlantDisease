# Plant Disease Web Application

This folder contains a simple web interface for running model inference on images.

Quick setup (development):

1. Create and activate your virtual environment.
2. Install project dependencies: `pip install -r requirements.txt`.
3. Start the demo backend (example):
   - Flask example (if implemented):

     ```bash
     flask run --app webapp.app
     ```

4. The web interface is currently minimal; see the `webapp` code for the specific framework in use (Flask/FastAPI/Streamlit).

If you'd like, I can add a small example endpoint to the webapp that runs the preprocessing demo on an uploaded image.
